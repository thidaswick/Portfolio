# Assets Folder

## Profile Image Instructions

To add your profile image:

1. **Place your image file here** with one of these names:
   - `profile.jpg` (recommended)
   - `profile.jpeg`
   - `profile.png`

2. **File location:**
   ```
   public/assets/profile.jpg
   ```

3. **Image requirements:**
   - Format: JPG, JPEG, or PNG
   - Recommended size: 800x800px or larger (square images work best)
   - The image will be automatically cropped to a circle

4. **After adding the image:**
   - Save the file
   - Refresh your browser (Ctrl + F5 for hard refresh)
   - The image should appear automatically
